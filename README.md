# Document Validator

## Requirements

- PHP 8.4
- Composer

## Installation

```bash
composer install
```

## Run script

```bash
php index.php
```

## Run tests

```bash
vendor/bin/phpunit
```

## Validation rules for tenants

config/tenant.rules.config.php


## DESCRIPTION

The project demonstrates the following design principles and patterns:

- Dependency Injection (DI)
- Factory Pattern
- Strategy Pattern
- SOLID principles:
    - Single Responsibility Principle (SRP)
    - Open/Closed Principle (OCP)

Unit tests are implemented using PHPUnit.
